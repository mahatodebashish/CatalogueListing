A Roku TV app to fetch the catalogue of videos from a restful API and display.

1. Screen shows various “categories” as fetched from the rest api.
2. Under the respective categories list of video cards is shown.
3. Every video card  display the “card” image and “title” fetched from the api data.
4. While fetching api data shows the progress indicator.
5. In case of network unavailability or error, shows error message and reload button.
6. Clicking on reload button, retry fetching data from network.
